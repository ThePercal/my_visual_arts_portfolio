The Portfolio

Description

The portfolio (this web), is a collection of all the activies I have made around all the year in visual arts class. I have choose to make it as a web.

Process

1. Firstly, I needed to create a folder for the proyect and open it in VS Code (the IDE and text editor I like).

2. Then I created the index.html file, the extension is .html because it is the programation language used for making webs. This file is going to be the "portrait" of the portfolio and the file which is going to make the web correctly.

3. After that I created proyectos.html, the file which will be what you see when clicking the botton of "See Proyects". It has the information to link each botton with its corresponding content.

4. Then I made so many files called proyecto1, proyecto2... as proyects are. Each file would have the information of each project.

5. Then I made the file style.css, this is for making the web pretty and give it colour, changing the founts of the text... The extension is called .css because css is the programtion language used for this.

6. I created a new folder to contain the photos, which was a called assest, as it is professionally called, and inside it is another folder called images, where the images are.

7. I started to write the code for all the files, while seeing if it works.

8. When the code was finished, to make the website public, I made a Github repository and uploaded the files.
